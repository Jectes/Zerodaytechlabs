# ZeroDay Tech Labs content plug ins

These JSON files are reusable content maps for the revised Start Here flow. They use icon names instead of emoji so the visuals stay consistent.

Files included:

* `start-here-content.json` maps the five home rules and emergency rule.
* `home-attack-types.json` maps the new attack types section.
* `case-study-plug-ins.json` maps case studies to home rules and attack types.
* `toolkit-hints.json` maps Toolkit hints to the same rule and attack flow.

Use these files when you want Case Studies, Toolkit cards, or future pages to point back to the Start Here rule and attack sections.
